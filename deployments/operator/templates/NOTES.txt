## Getting Started

1. **Verify the operator is running:**
   kubectl get deployment {{ include "jira-sync-operator.fullname" . }} -n {{ include "jira-sync-operator.namespace" . }}

2. **Check operator logs:**
   kubectl logs -n {{ include "jira-sync-operator.namespace" . }} -l app.kubernetes.io/name={{ include "jira-sync-operator.name" . }} -f

3. **View available CRDs:**
   kubectl get crd | grep sync.jira.io

## Create Your First JIRASync Resource

Create a JIRA credentials secret:
kubectl create secret generic jira-credentials \
  --from-literal=base-url=https://your-domain.atlassian.net \
  --from-literal=email=your-email@company.com \
  --from-literal=token=your-personal-access-token

Then create a JIRASync resource:
cat <<EOF | kubectl apply -f -
apiVersion: sync.jira.io/v1alpha1
kind: JIRASync
metadata:
  name: my-first-sync
spec:
  jiraConfig:
    baseURL: "https://your-domain.atlassian.net"
    credentialsSecret: "jira-credentials"
  syncConfig:
    mode: "Single"
    issueKeys: ["PROJ-123"]
  gitConfig:
    repository: "/tmp/my-sync-repo"
EOF

## Monitor Sync Progress

Watch your sync:
kubectl get jirasync my-first-sync -w

Check sync status:
kubectl describe jirasync my-first-sync
